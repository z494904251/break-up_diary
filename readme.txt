2018/01/07 广州 小雨
今天呢。刚好是元旦过后的第一个星期。你上次找我就是元旦的那天，晚上九点多。收到你信息还挺开心的。那时候在打游戏，还是挺惊讶的，毕竟你手机把我拉黑了还能收到你的信息。回你之后也没说什么了，不知道你有没有收到。至少现在这种情况下，我知道我可能慢慢地不能再像以前那样子去找你了。之前找你的时候，你的一些举动也真的让我觉得突然一下子就很有了距离感。想到之前的你就会让我心里很不舒服。那段时间就会经常打游戏来麻木自己。但是你跟我说，“你要变得更好”。对呀。我也觉得我一定要变得更好。这好几天我还看了很多的材料，又新学了很多东西，还把《心里的声音》看完了。这部剧还真的挺好看的。最后的那句话也启发了我很多，让我知道应该用一种怎样的态度来处理自己的梦想，我发了朋友圈，不知道你有没有看见。很谢谢你。
